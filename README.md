# smart-crosswalk

> 2026-1 산업시스템공학종합설계 (Senior Project in ISE) | 8조

고령 보행자 사고 예방을 위한 보행신호 자동연장 스마트 횡단보도 최적 설치 입지 선정

---

## 프로젝트 개요

한국 교통사고 사망자 중 고령자 비율은 51.5%, 그중 보행 사망의 67%가 65세 이상이다.
현행 보행신호는 1.0m/s 기준으로 설계되어 있으나, 고령자 대다수는 이 속도를 충족하지 못한다.
보행신호 자동연장(카메라/센서 기반)은 이 구조적 갭을 해결하는 검증된 수단이지만,
개소당 3,000만~8,000만 원의 비용 제약으로 전수 설치가 불가능하다.

**본 연구는 서울시 중구 1,098개 횡단보도를 대상으로,
사고 위험도와 교통 영향을 동시에 고려한 최적 설치 입지 선정 모델을 제안한다.**

---

## 방법론 파이프라인
[1단계] 데이터 수집·전처리 ← 완료
[2단계] ML: 사고 위험 점수
[3단계] 대기행렬: 교통 영향 점수
[4단계] 정수계획법: 최적 설치 조합 (사고위험↑ + 교통영향↓ trade-off)
[5단계] SUMO 시뮬레이션: 검증 + 전략 비교

text

| 단계 | 방법론 | 역할 |
|------|--------|------|
| 2 | 수요예측 (ML) | 횡단보도별 사고 위험 점수 산출 |
| 3 | 대기행렬 (Queueing) | 신호 연장 시 차량 지체 점수 산출 |
| 4 | 정수계획법 (IP) | 예산 제약 하 최적 설치 조합 선정 |
| 5 | 시뮬레이션 (SUMO) | 설치 전/후 효과 비교 검증 |

## 평가 전략

| 전략 | 설명 |
|------|------|
| 전략 1 | 사고 건수 순 설치 (단순 랭킹) |
| 전략 2 | 고령 인구 많은 행정동 우선 |
| 전략 3 | 랜덤 설치 (민원 기반 대리) |
| **전략 4** | **ML + IP 최적화 (제안 모델)** |

| 지표 | 의미 |
|------|------|
| 위험 커버율 | 설치 지역 예측 사고 합 / 전체 예측 사고 합 |
| 사고 감소 기대량 | 예측 사고 × 효과율 (해외 24%) |
| 비용 대비 효율 | 위험 커버 총점 / 총 설치 비용 |
| 전략 간 개선율 | 제안 모델 vs 단순 랭킹 차이 |
